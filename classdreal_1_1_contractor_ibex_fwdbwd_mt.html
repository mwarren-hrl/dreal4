Detailed Description

Multi-thread version of ContractorIbexFwdbwd contractor.

The base ContractorIbexFwdbwd is not thread-safe. When there are N jobs, it creates N ContractorIbexFwdbwd instances internally and make sure that each thread calls a designated instance.
