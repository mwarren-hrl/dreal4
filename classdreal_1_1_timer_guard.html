Detailed Description

Pauses the passed timer object when the guard object is destructed (e.g.

going out of scope).

Constructs the timer guard object with timer.

If enabled is false, this class does not do anything. If start_timer is true, starts the timer in the constructor. Otherwise, it does not start it and a user has to call resume() to start it.
